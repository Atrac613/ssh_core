# Code of Conduct

This project aims to be technically rigorous, kind, and easy to collaborate on.

## Our Expectations

Please:

- be respectful in technical disagreements
- assume good intent and ask clarifying questions before escalating
- keep feedback specific, actionable, and tied to the code or behavior
- welcome contributors with different backgrounds and experience levels
- avoid sarcasm, hostility, harassment, or personal attacks

## Scope

These expectations apply to issues, pull requests, code review, discussions,
and other project spaces tied to `ssh_core`.

## Maintainer Response

Project maintainers may edit, remove, or moderate comments and contributions
that make collaboration unsafe or unproductive.

## Reporting

If you run into behavior that makes participation difficult, open a private
channel with the maintainers instead of continuing the conflict in public.
